# 자동차 경주 게임
## 진행 방법
* 숫자 야구 게임 요구사항을 파악한다.
* 요구사항에 대한 구현을 완료한 후 자신의 github 아이디에 해당하는 브랜치에 Pull Request(이하 PR)를 통해 과제를 제출한다.

##요구사항
* 자동차 이름/거리 입력할 수 있는 객체 생성
* 자동차 생성하는 로직 구현
* 자동차 역할에 대한 객체 생성
    * 자동차 움직이는 로직 구현
        * 랜덤 숫자 생성
        * 숫자에 따른 전진 로직 구현
    * 자동차 움직이는 프린트 method 구현
    * 가장 멀리 이동한 거리 구하는 로직 구현
*거리에 따른 자동차 이름 얻어내는 로직 구현
